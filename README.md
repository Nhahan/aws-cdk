# aws-cdk

run commands on the directory that has actual IaC resources


### init
```
cdk init app --language=typescript
```

### cdk documentation
```
cdk doc
```

### make templates
```
cdk synthesize --output=./templates
```

### show the difference between the what's deplyed and what we have on our code
```
cdk diff
```

### deploy
```
cdk deploy
```

### destroy
```
cdk destroy
```
